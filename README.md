I found out that we don't have a repository and that code tends to stay with a project. This is a pity for a few reasons. 
1) It's difficult to find, especially if it is old.
2) It's vulnerable to being accidentally deleted or changed
3) There is no version control,
4) There is limited ability to create branches and test ideas.

I created this in response to the perceived need. Feel free to use the repository however you like. Please try to keep projects in sub-folders so we can find things.
